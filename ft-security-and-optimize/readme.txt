== Description ==
A single admin menu that lets you toggle essential security and performance features of WordPress with checkboxes—hide the WordPress version, disable XML‑RPC, REST‑API, JSON, Heartbeat, DNS‑Prefetch, RSD link, emojis, and embeds. Ideal for fast, clean hardening and optimization of your site.

== Installation ==

1. Upload `ft-security-and-optimize` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
